============
Installation
============

.. contents::
   :local:

You can install scimma-slack-app via pip or from source.

To install with pip:

.. code:: bash

   pip install -U scimma-slack-app

To install from source:

.. code:: bash

    tar -xzf scimma-slack-app-x.y.z.tar.gz
    cd scimma-slack-app-x.y.z
    python setup.py install
