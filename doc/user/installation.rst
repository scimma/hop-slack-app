============
Installation
============

.. contents::
   :local:

You can install hop-slack-app via pip or from source.

To install with pip:

.. code:: bash

   pip install -U hop-slack-app

To install from source:

.. code:: bash

    tar -xzf hop-slack-app-x.y.z.tar.gz
    cd hop-slack-app-x.y.z
    python setup.py install
